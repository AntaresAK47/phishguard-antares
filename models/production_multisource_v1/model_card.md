# Model Card — production_multisource_v1

Modelos entrenados con SpaPhish, SpearPhishMX, PhishTank y Tranco.

## Uso metodológico

- Textual global: SpaPhish + SpearPhishMX.
- URL global: PhishTank + Tranco.
- Fusión híbrida: combina probabilidad textual y probabilidad URL.
- Las URLs se procesan como texto; no se visitan enlaces.
